# OpenCV Basics Practice

This repository contains my beginner-level OpenCV practice programs written in Python. It covers fundamental image processing techniques and webcam access using OpenCV.

## 📂 Project Structure

```
.
├── basic.py
├── webcampractice.py
├── image.jpeg
└── README.md
```

---

## 📌 Features Covered

### `basic.py`

This file demonstrates the following OpenCV concepts:

* Reading an image
* Displaying image properties
* Converting BGR image to Grayscale
* Gaussian Blur
* Binary Thresholding
* Binary Inverse Thresholding
* Adaptive Gaussian Thresholding
* Median Blur
* Bilateral Filter
* Sobel Edge Detection (X & Y)
* Canny Edge Detection
* Morphological Operations

  * Erosion
  * Dilation
  * Opening
  * Closing
* Drawing a Line on an Image
* Displaying multiple windows using `cv2.imshow()`

---

### `webcampractice.py`

This file demonstrates:

* Accessing the default webcam
* Reading live video frames
* Displaying webcam feed in real time
* Exiting the webcam using the **Q** key

---

## 🛠️ Requirements

* Python 3.x
* OpenCV
* NumPy

Install the required packages:

```bash
pip install opencv-python numpy
```

---

## ▶️ Running the Programs

### Run Image Processing Demo

```bash
python basic.py
```

Make sure `image.jpeg` is present in the same directory.

### Run Webcam Demo

```bash
python webcampractice.py
```

Press **Q** to close the webcam window.
